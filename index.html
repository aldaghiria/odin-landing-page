Git Basic Commands

git status

The git status command is used to display the state of the repository and staging area.

git add .

take a snapshot of the contents of all files under the current directory

git commit

The git commit is used to store the snapshot temporary staging area which Git calls the "index". 

git push

The git push command is used to upload local repository content to a remote repository
